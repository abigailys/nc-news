
# 📰 NC News Frontend

## Overview

NC News is a responsive React application that consumes the NC News API. It allows users to browse, read and interact with news articles in a Reddit-style platform.

Users can view articles by topic, read individual articles, vote on content and post comments. The application focuses on clear UI structure, responsive design and smooth user interaction.

This project was built as part of the Northcoders Software Development Bootcamp.

## Features

- Browse articles by topic
- View individual articles with comment threads
- Vote on articles with optimistic UI updates
- Post new comments
- Delete comments (where permitted)
- Client-side routing
- Loading and error states
- Responsive layout

## Key Concepts Demonstrated

- React component architecture
- State management using React Hooks
- API integration using Axios
- Client-side routing with React Router
- Conditional rendering
- Optimistic UI updates
- Basic accessibility and performance considerations

## Getting Started Locally

### 1. Clone the repository

    git clone https://github.com/abigailys/nc-news.git
    cd nc-news

### 2. Install dependencies

    npm install

### 3. Run the development server

    npm run dev

The app should now be running locally.
